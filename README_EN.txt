This is Moldavian LibreOffice dictionary extension.
This extension implements the classic Moldavian orthography (î/sînt).
The package contains hyphenation, spelling dictionaries and a thesaurus.
This archive was put toghether by Mihai Vasiliu (https://github.com/SilverGreen93/dict-md)

Based on rospell version 1.7.
